Dernières versions

  • v1.4 - Anti-Hallucination

    Le prompt du modèle de réponse a été renforcé pour empêcher l'invention d'informations lorsque les données sont manquantes.

  • v1.3 - Amélioration des dates

    Le prompt de génération SQL a été rendu plus strict pour forcer l'utilisation de `date_of_inventory` et ignorer les dates nulles.

  • v1.2 - Correction de bugs Mistral

    Correction des erreurs de crash avec le modèle Mistral en utilisant une version stable de la librairie et en adaptant les appels API.

  • v1.1 - Ajout de Mistral AI

    Ajout d'un sélecteur de modèle pour permettre de choisir entre OpenAI (GPT-4o-mini) et Mistral (Mistral Small).

  • v1.0 - Version initiale

    Première version du chatbot d'inventaire utilisant le modèle d'OpenAI pour générer des requêtes SQL.
